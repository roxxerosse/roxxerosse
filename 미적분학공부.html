극한 (Limits)

함수의 극한 개념과 성질을 학습하는 섹션입니다.

극한의 정의

함수 $f(x)$에서 $x$가 $a$에 가까워질 때, $f(x)$의 값이 특정 실수 $L$에 가까워지면, "$x \to a$일 때 $f(x)$의 극한은 $L$이다"라고 하고 다음과 같이 나타냅니다:

$$\lim_{x \to a} f(x) = L$$

극한의 존재 조건:

  • 좌극한: $\lim_{x \to a^-} f(x)$
  • 우극한: $\lim_{x \to a^+} f(x)$
  • 극한이 존재 ⟺ 좌극한 = 우극한

미분 (Derivatives)

함수의 미분 개념과 미분 규칙을 학습하는 섹션입니다.

미분의 정의

함수 $f(x)$가 $x = a$에서 미분가능하면, $f$의 $x = a$에서의 미분계수는:

$$f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h}$$

미분 규칙:

  • 곱의 미분: $(uv)' = u'v + uv'$
  • 몫의 미분: $\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}$
  • 합성함수의 미분: $(f \circ g)'(x) = f'(g(x)) \cdot g'(x)$

적분 (Integrals)

부정적분과 정적분의 개념을 학습하는 섹션입니다.

부정적분

함수 $f(x)$의 부정적분이란, 미분하면 $f(x)$가 되는 함수 $F(x)$를 찾는 것입니다:

$$\int f(x)dx = F(x) + C \quad \text{(단, } F'(x) = f(x) \text{)}$$

적분 규칙:

  • $\int x^n dx = \frac{x^{n+1}}{n+1} + C$ (단, $n \neq -1$)
  • $\int e^x dx = e^x + C$
  • $\int \sin x dx = -\cos x + C$
  • $\int \cos x dx = \sin x + C$
